## Clone Enviroments

## Setup
1. Clone this repository  
2. ./dotfilesLink.sh  
3. git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
4. vim
5. :BundleInstall
